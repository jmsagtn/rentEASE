Data Encryption

Encryption is at the heart of our security infrastructure. We use multiple layers of encryption to protect your data:

In Transit

  • TLS 1.3: Latest transport layer security protocol for all data transmission
  • 256-bit SSL Certificates: Bank-level encryption for data moving between your device and our servers
  • Perfect Forward Secrecy: Each session uses unique encryption keys that can't be compromised retroactively
  • Certificate Pinning: Protection against man-in-the-middle attacks

At Rest

  • AES-256 Encryption: Military-grade encryption for all stored data
  • Encrypted Backups: All backup copies are encrypted with separate keys
  • Database Encryption: Multiple layers of encryption at the database level
  • Secure Key Management: Encryption keys stored separately from encrypted data

Backup & Disaster Recovery

We maintain comprehensive backup and disaster recovery procedures to ensure your data is never lost:

üîê Password Security

  • Use a strong, unique password with at least 12 characters

  • Consider using a reputable password manager
